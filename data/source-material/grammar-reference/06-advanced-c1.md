# Grammaire avancée C1 — Référence
Source : Kramer ch.9-16, Lunt §13+

## 1. NOMS VERBAUX (глаголски именки) — ch.9
Formation : L-forme pl. (-le) → remplacer -le par -nje/-enje
- chitanje (la lecture), pevanje (le chant), jadenje (manger)
- rabotenje (le travail), uchenje (l'apprentissage)
- zboruvanje (le fait de parler)

Genre : neutre. Article : -to → chitanjeto
Pluriel rare : zanimanja (passe-temps) — collectif

Usage :
- Chitanjeto e vazno. (La lecture est importante.)
- Sakam chitanje. (J'aime lire — litt. j'aime la lecture.)

## 2. VOCATIF — interpeller quelqu'un
Masculins en consonne + noms propres : ajout de -e
- Branko → Branke! / Pero → Pere! / Marko → Marko! (pas de changement si -o)
- Chovek → chovece! (petit homme / mon homme)
- Brate! / Sine! / Druzhе! (Frère! / Fils! / Ami!)

Féminins en -a : -a reste (ou -o dans certains dialectes)
- Majko! / Sestra! / Biljana! (souvent pas de changement)

Usage : interpellation affectueuse, pas obligatoire en standard

## 3. DIMINUTIFS ET AUGMENTATIFS
Suffixes diminutifs courants :
- Masc. -che : grad → gradche (petite ville), sin → sinche
- Fém. -ica : kuka → kukicka (petite maison)
- Neutre -nce : dete → detence (tout petit enfant)

Suffixes augmentatifs (+ expressifs) :
- -ina : nosina (grand nez), chovechina (gros bonhomme)

## 4. PASSIF avec SE
Le passif s'exprime généralement avec se + verbe :
- Toj jazik se zboruva vo Makedonija. (Cette langue est parlée en Macédoine.)
- Pismoto se pishuva utre. (La lettre sera écrite demain.)
- Nashol se resenie. (Une solution a été trouvée.)

Passif avec "bian" (formel/littéraire) :
- Pismoto e napisano. (La lettre a été écrite.) — adjectif verbal passif

## 5. ADMIRATIF (изјавен начин) — action rapportée/surprise
Expression d'étonnement ou de doute sur ce qu'on rapporte.
Formé avec ke + L-forme :

- Toj ke bil vo Ohrid! (Il serait à Ohrid ! / Il paraît qu'il est à Ohrid!)
- Tie ke imale pari! (Ils auraient de l'argent, paraît-il!)

Très courant en macédonien oral pour distancer une information.

## 6. CONDITIONNEL (условен начин)
### Conditionnel présent (souhait/possibilité)
bi + L-forme :
- Jas bi otishol. (J'irais. / Je voudrais aller.)
- Nie bi imale vreme. (Nous aurions le temps.)

### Conditionnel avec ako (si)
ako + imparfait → bi + L-forme :
- Ako imav pari, bi kupal kola. (Si j'avais de l'argent, j'achèterais une voiture.)
- Ako znael, bi kazhal. (S'il avait su, il aurait dit.)
- Ako ne bev bolen, ke/bi otishol. (Si je n'avais pas été malade, je serais allé.)

## 7. PROPOSITIONS RELATIVES
### Avec shto (invariable) — complément relatif
- Chovekot shto go videv e tuka. (L'homme que j'ai vu est ici.)
- Knigata shto ja chitam e interesna. (Le livre que je lis est intéressant.)

### Avec koj/koja/koi (accordé) — sujet relatif
- Zhenata koja dojde e moja prijatelka. (La femme qui est venue est mon amie.)
- Lugjet koi rabotаt tука se od Makedonija. (Les gens qui travaillent ici sont de Macédoine.)

### Chij (dont, possessif relatif) — rare mais important C1
- Chovekot chijot sin stüdira vo Pariz... (L'homme dont le fils étudie à Paris...)
- Zhenata chijata kuka e golema... (La femme dont la maison est grande...)

## 8. QUESTIONS INDIRECTES (subordonnées interrogatives)
- Dali znaesh kade zhivee? (Sais-tu où il habite?)
- Ne znam koj dojde. (Je ne sais pas qui est venu.)
- Me pra shaa koga ke bidam doma. (Ils m'ont demandé quand je serais chez moi.)
- Reci mi shto sakas. (Dis-moi ce que tu veux.)

## 9. VERBES DE SENTIMENT (sacanje/dopaganje)
Verbes qui prennent un OBJET INDIRECT comme sujet grammatical :
- Mi se saka kafe. (J'ai envie d'un café — litt. le café me se-veut.)
- Mu se jade. (Il a faim — litt. le manger lui se-mange.)
- Ne mi se odi. (Je n'ai pas envie d'aller.)
- Mi se spie. (J'ai sommeil.)
- Ti se dopaǵa muzika? (Est-ce que tu aimes la musique? — litt. la musique te plaît?)

## 10. ASPECT DANS LES PROPOSITIONS TEMPORELLES
- Dodeka (pendant que) + imperfectif : Dodeka chitam, toj pee. (Pendant que je lis, il chante.)
- Otkako (depuis que/après que) + perfectif : Otkako zavrshe, otide. (Après qu'il eut fini, il repartit.)
- Pred da (avant de) + perfectif présent : Pred da otide, mu kaza. (Avant de partir, il lui dit.)

## 11. PARTICIPE ADJECTIVAL PASSIF (-en/-an)
Formé depuis le radical perfectif :
- napísha → napíshen (écrit)
- napravi → napravеn (fait)
- ochi sta → ochistеn (nettoyé)

Accord : napíshen/napíshena/napísheno/napísheni
Usage : kukata е napravena (la maison est faite/construite)

## 12. PRÉPOSITIONS AVEC PRONOMS
Après préposition, toujours forme FORTE :
- od mene (de moi), so nego (avec lui), za nea (pour elle)
- pri nas (chez nous), kон nim (vers eux)
- za sebe (pour soi — réfléchi)
