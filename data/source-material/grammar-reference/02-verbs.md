# Verbes macédoniens — Référence complète
Source : Kramer ch.2-12, Lunt §5-12, Kepeski

## 1. GROUPES DE CONJUGAISON (3 groupes + irréguliers)

### Identification : voyelle de la 3e sg présent
- Groupe A (-a) : saka, gleda, zboruva, chita, raboti (certains)
- Groupe I (-i) : nosi, vodi, pravi, odi
- Groupe E (-e) : jade, pije, bere, lezhe

### Présent — terminaisons
| Personne | Groupe A | Groupe I | Groupe E |
|---|---|---|---|
| Jas | -m (sakam) | -am (nosam) | -am (jadam) |
| Ti | -sh (sakash) | -ish (nosish) | -sh (jadesh) |
| Toj/Taa | ø (saka) | ø (nosi) | ø (jade) |
| Nie | -me (sakame) | -ime (nosime) | -me (jademe) |
| Vie | -te (sakate) | -ite (nosite) | -te (jadete) |
| Tie | -aat (sakaat) | -at (nosat) | -at (jadat) |

**Irréguliers présent** : sum (sum/si/e/sme/ste/se), znae→znam, stoi→stojam, pije→pijam

## 2. IMPARFAIT (passé imperfectif, continu)

Formation : radical + terminaisons selon groupe
| Personne | Groupe A (gleda) | Groupe I (nosi) | Groupe E (jade) |
|---|---|---|---|
| Jas | gledav | nosev | jadev |
| Ti | gledase | nosese | jadese |
| Toj/Taa | gledase | nosese | jadese |
| Nie | gledavme | nosevme | jadevme |
| Vie | gledavte | nosevte | jadevte |
| Tie | gledaa | nosea | jadea |

**Sum imparfait** : bev/bese/bese/bevme/bevte/bea

## 3. AORISTE (passé perfectif, action complète vécue)

Terminaisons IDENTIQUES pour tous les groupes :
| Personne | Terminaison |
|---|---|
| Jas | -v |
| Ti | ø |
| Toj/Taa | ø |
| Nie | -vme |
| Vie | -vte |
| Tie | -a |

### Groupe A (radical conservé) : prochita
prochitav / prochita / prochita / prochitavme / prochitavte / prochitaa

### Groupe I : napravi
napraviv / napravi / napravi / napravivme / napravivte / napravija (-ija !)

### Groupe E (e→a) : kazhe
kazhav / kazha / kazha / kazhavme / kazhavte / kazhaa

### Groupe E-O (e conservé à 2e/3e sg) : dojde, otide, donese
dojdov / dojde / dojde / dojdovme / dojdovte / dojdoa

**Irréguliers** :
- dade : dadov/dade/dade/dadovme/dadovte/dadoa
- vide : vidov/vide/vide/vidovme/vidovte/vidoa
- rece : rekov/reche/reche/rekovme/rekovte/rekoa
- sum : bev/beshe/beshe/bevme/bevte/bea (= imparfait)

## 4. L-FORMES (глаголска л-форма) 

### Formation : aoriste 1sg → suppr. -v → ajout -l/-la/-lo/-le
| Forme | Terminaison | Exemple (napravi) |
|---|---|---|
| Masc. sg | -l | napravil |
| Fém. sg | -la | napravila |
| Neutre sg | -lo | napravilo |
| Pluriel | -le | napravile |

### Irréguliers L-formes
| Présent | L-forme (m./f./pl.) |
|---|---|
| dojde | doshol/doshla/doshle |
| otide | otishol/otishla/otishle |
| najde | nashol/nashla/nashle |
| pojde | poshol/poshla/poshle |
| dade | dal/dala/dale |
| izede | izel/izela/izele |
| rece | rekol/rekla/rekle |
| vide | videl/videla/videle |
| sum | bil/bila/bile |
| donese | donel/donela (préféré) ou donesol |

### Passé composé avec sum + L-forme
Jas sum orel/orela / Ti si orel/orela / Toj e orel / Taa e orela /
Nie sme orele / Vie ste orele / Tie se orele
Note : À la 3e sg. l'auxiliaire 'e' est souvent omis en conversation.

## 5. FUTUR

### Futur affirmatif : ke + présent
ke chitam, ke chitash, ke chita, ke chitame, ke chitate, ke chitaat

### Futur avec clitiques : ke + dat + acc + verbe
Ke mi go donese. / Ke ja prochitam.

### Futur négatif : nema + da + présent (OU ne + ke + verbe)
Nema da chitam. = I ne ke chitam.

### Futur de sum : ke + bide (irrégulier !)
ke bidam / ke bidesh / ke bide / ke bideme / ke bidete / ke bidat

## 6. IMPÉRATIF

Formation depuis 3e pl présent → supprimez -at/-aat → ajout -i (sg) ou -ete (pl)
OU pour verbes en -uva : -aj (sg) / -ajte (pl)

| Verbe | Impératif sg | Impératif pl |
|---|---|---|
| odi (aller) | odi! | odete! |
| jade (manger) | jadi! | jadete! |
| kazhe (dire) | kazhi! | kazhete! |
| zboruva (parler) | zboruvaj! | zboruvajte! |
| dojde (venir) | dojdi! | dojdete! |
| dade (donner) | daj! | dajte! |
| vidi (voir) | vidi! | videte! |
| se mie (se laver) | izmij se! | izmijte se! |

**Impératif négatif** : ne + impératif
Ne odi tamu! / Ne jadi toa! / Ne go kazhi!

**Impératif 3e pers.** : neka + présent
Neka dojde! (Qu'il vienne!) / Neka jadaat! (Qu'ils mangent!)

## 7. CONSTRUCTIONS AVEC DA (= infinitif macédonien)

`verbe modal + da + verbe conjugué au présent`

| Modal | Sens | Conjugaison |
|---|---|---|
| mozhe | pouvoir | mozham/mozhesh/mozhe/mozheme/mozhete/mozhaat |
| treba | devoir (nécessité) | treba (souvent invariable) |
| mora | devoir (obligation) | se conjugue normalement |
| saka | vouloir | sakam/sakash/saka/sakame/sakate/sakaat |
| pochnuva | commencer à | groupe A |
| prestanuva | arrêter de | groupe A |
| moze li da | demande permission | invariable |

Clitiques dans la subordonnée da : avant le verbe (pas avant le modal)
- Saka da mu kazhe. (Il veut lui dire.) ✓
- *Mu saka da kazhe. ✗

## 8. CONDITIONNEL

Deux types :
### Conditionnel présent : bi + L-forme
(Exprime possibilité ou souhait non réalisé)
- Jas bi sakал да учам. (Je voudrais apprendre.)
- Nie bi otishle. (Nous serions partis.)

### Conditionnel avec ako (si) :
ako + imparfait → bi + L-forme
- Ako imav vreme, bi otishol. (Si j'avais le temps, j'irais.)
- Ako znael, bi kazhal. (S'il avait su, il aurait dit.)

## 9. NOMS VERBAUX (глаголски именки)

Formation : L-forme pl. → remplacement -le par -nje
- chita → chitanje (la lecture)
- peva → pevanje (le chant)  
- raboti → rabotenje (le travail)
- jade → jadenje (l'action de manger)
- zboruva → zboruvanje (le fait de parler)

Genre : neutre (terminaison -nje ou -enje)
Article défini : -to → chitanjeto (la lecture)
