# Aspect verbal — Référence complète
Source : Kramer ch.6/8/9, Lunt §5, Kepeski

## Définition
Chaque verbe macédonien appartient à l'un des deux aspects :
- **IMPERFECTIF** (nesvrsheno) : action en cours, répétée, générale, sans fin définie
- **PERFECTIF** (svrsheno) : action complète, ponctuelle, avec résultat visible

## Règles d'usage

| Situation | Aspect | Exemple |
|---|---|---|
| Action en cours maintenant | Imperfectif | Sega chitam (je lis maintenant) |
| Habitude répétée | Imperfectif | Sekoj den chitam (je lis chaque jour) |
| Action générale/description | Imperfectif | Mnogu sakate da zboruvate (vous aimez parler) |
| Action unique et complète | Perfectif | Ja prochitav knigata (j'ai lu le livre en entier) |
| Début d'action | Perfectif | Sakaаt da zapeat (ils veulent commencer à chanter) |
| Résultat obtenu | Perfectif | Napishi mu (écris-lui — et finis-le) |
| Séquence d'événements passés | Perfectif | Dojde, jade, otide (il vint, mangea, repartit) |

## Formes par aspect et temps
| Temps | Imperfectif | Perfectif |
|---|---|---|
| Présent | ✓ (chita) | ✗ (doit être précédé de da/ke) |
| Imparfait | ✓ (chitase) | ✓ (ke prochitase — peu usité) |
| Aoriste | ✓ (chitav — rare) | ✓ (prochitav) |
| Futur | ke chita (en cours futur) | ke prochita (complété dans le futur) |
| L-forme | chital (habituel/rapporté) | prochital (complété/rapporté) |
| Impératif | chitaj! (lire en général) | prochitaj! (lire et finir) |

## Paires aspectuelles — Liste des plus courantes

### Type 1 : imperfectif non-préfixé / perfectif préfixé
| Imperfectif | Perfectif | Sens |
|---|---|---|
| pee | zapee | chanter / commencer à chanter |
| chita | prochita | lire / lire jusqu'au bout |
| pravi | napravi | faire / faire et finir |
| pishuva | napíshe | écrire / finir d'écrire |
| govori | progovori | parler / prendre la parole |
| gleda | pogledа | regarder / jeter un coup d'œil |
| barie | pobara | chercher / trouver après avoir cherché |

### Type 2 : différence de suffixe (-uva = imperfectif)
| Imperfectif | Perfectif | Sens |
|---|---|---|
| kazhuva | kazhe | dire |
| kupuva | kupi | acheter |
| pochenuva | pochne | commencer |
| slushuva | slushne | entendre |
| isprakja | isprati | envoyer |
| donésuva | donese | apporter |

### Type 3 : préfixe ET suffixe différents
| Imperfectif | Perfectif | Sens |
|---|---|---|
| pishuvа | napíshe | écrire |
| pokаzhuvа | pokаzhe | montrer |

### Type 4 : verbes entièrement différents (suppletifs)
| Imperfectif | Perfectif | Sens |
|---|---|---|
| gleda | vidi | regarder / voir |
| slüsha | chue | écouter / entendre |
| veli | rece | dire (habituellement) / dire (une fois) |
| zboruva | prozbori | parler / prendre la parole |
| odi | otide | aller (processus) / arriver (résultat) |
| doaǵa | dojde | venir (processus) / arriver (résultat) |

### Type 5 : même verbe = les deux aspects (rares)
- chuva (garder), pita (demander), celiva (embrasser)
- Verbes en -ira : telefoniram (imperfectif ET perfectif)

## Aspect et KOGA (quand)
- koga + imperfectif = PENDANT QUE (simultané)
- koga + perfectif = QUAND (séquentiel)

Koga ти ke chitash (imp.), nie ke gledame televizija.
→ Pendant que tu liras, nous regarderons la télé.

Koga ke ja prochitam (pf.) knigata, ke ti ja dаdam.
→ Quand j'aurai fini de lire le livre, je te le donnerai.

## Verbes de mouvement — système aspectuel particulier
| Imperfectif processus | Perfectif résultat |
|---|---|
| odi (aller, processus) | otide (partir et arriver) |
| doaǵa (venir, processus) | dojde (arriver) |
| izheguva (sortir, hab.) | izleze (sortir une fois) |
| vlegуva (entrer, hab.) | vleze (entrer une fois) |
| minuva (passer, hab.) | pomíne (passer une fois) |
